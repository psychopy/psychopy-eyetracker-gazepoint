-------------------------------
GazepointEyetrackerBackend
-------------------------------

An eyetracker device backend for GazePoint eyetrackers


    **Categories:** Custom
    **Works in:** PsychoPy

Parameters
-------------------------------

Eyetracking
=================

GazePoint IP address
    IP Address of the computer running GazePoint Control.

GazePoint port
    Port of the GazePoint Control server. Usually 4242.
